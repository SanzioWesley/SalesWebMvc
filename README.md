<h1>SalesWebMvc</h1>

Aplicação Web para simular uma lista de vendedores e seus departamentos, onde é possível criar, editar e excluir as vendas.

## :wrench: Tecnologias

 - Bootstrap
 - c#
 - Entity Framework Core
 - MySql
